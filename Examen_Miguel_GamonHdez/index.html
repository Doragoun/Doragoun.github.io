Comandos de Linux


Comando para Listar Carpetas y Documentos

~$ ls
el comando ls te permite enumerar todos los archivos o directorios en el directorio actual de forma predeterminada, e interactuar con ellos a través de la línea de comandos.

Opcion -l del comando ls
~$ ls-l
Este comando nos muestra el contenido del directorio en un formato de tabla con las columnas incluidas:
  • Permisos de contenido
  • Numero de enlaces al contenido
  • Propietario del contenido
  • propietario del grupo del contenido
  • tamaño del contenido en bytes
  • fecha / hora de la última modificación del contenido
  • nombre de archivo o directorio

Opcion -a del comando ls
~$ ls-a
Escribe el comando ls -a para listar archivos o directorios, incluidos archivos o directorios ocultos. En Linux, cualquier cosa que comience con un . se considera un archivo oculto.

Comandos para crear un documento

~$ touch nombre_del_archivo
Uno de los mayores usos del comando touch en Linux es crear un nuevo archivo vacío.
Si el archivo no existe todavía, creará un nuevo archivo vacío. Si ya existe un archivo con el mismo nombre, actualizará las marcas de tiempo del archivo.

Comandos para crear un directorio

~$ mkdir nombre_del_directorio
El comando mkdir se utiliza para crear nuevos directorios en Linux.

Comandos para eliminar documentos

~$ rm nombre_del_documento
Este comando se utiliza para eliminar el archivo con el nombre deseado

Opcion para eliminar directorios
~$ rm -r Nombre_del_Directorio
Usa una opción recursiva, representada por -r, para eliminar el directorio y su contenido.

Comando para modificar un documento con nano

~$ nano Nombre_del_archivo
