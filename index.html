VIM

Este es un editor de texto mas minimalista
El comando para acceder a este es:
~$ vim Nombre_del_archivo

NANO

Este es un editor de texto mas simple y poco mas sencillo de usar.
Para abrir el editor de texto se utiliza el siguiente comando:
~$ nano Nombre_del_archivo
Algunos de sus comandos utilizados para la utilizacion de este son:
  • ^X Exit
  • ^R Read File
  • ^O Write Out

MICRO

Este es un editor de texto ya mas orientado a lo que conocemos ahorita como podria ser Microsoft Word
Para abrir el editor de texto se utiliza el siguiente comando:
~$ micro Nombre_del_archivo
Algunos de los comandos que se utilizan en este editor son:
  • ^S Save
  • ^Q Exit
  • ^C Copy
  • ^V Paste
  • ^X Cut
  • ^B Nos permite ejecutar un comando bash
